Our Mission

Mathematics is often taught in terms of symbols on paper. PrimeVerse is founded on the belief that numbers have stories and patterns that are best understood when explored interactively. By combining premium Glassmorphic frontend design systems with responsive calculators, we aim to bridge the gap between static content and engaging, exploratory science.

Technologies Used

HTML5 CSS3 Bootstrap 5 Vanilla JavaScript (ES6+) Local Storage API AOS Animation Library KaTeX Formula Renderer

Project Highlights

Interactive Calculations

Factor trees, range sieves, and primality checks calculated instantly inside the browser.

HSL Variable Themes

Light/Dark modes swap instantly using synchronized custom CSS properties and store in Local Storage.

Bookmark & Likes Storage

Save your favorite articles and track your liked articles dynamically using the browser's storage.

Local Comment Boards

Participate in discussions by writing comments that persist across session reloads.

Skills Demonstrated

  • Responsive Fluid Grid Layouts: Implemented Bootstrap flex systems to restructure layout columns seamlessly across widescreen desktop, tablet, and mobile screens.
  • State Management in Plain JS: Coordinate multiple components (likes, bookmarks, theme selectors, search indexes) using custom event dispatchers.
  • Custom CSS Animation Controls: Design performant transitions, keyframed blobs, and card translations using optimized hardware-accelerated transforms.
  • Math typesetting integration: Utilize KaTeX CDN pipelines to format algebraic calculations and LaTeX notations dynamically.
